核心格言

「先看病人，再看波形，最後才解讀數字。」

異常數據評估的第一原則

「先排除假的異常，再處理真的休克。」

避免根據錯誤監測數據做出錯誤治療

「ProAQT 主要看趨勢，不要迷信單點數字。」

ProAQT 是趨勢監測工具，不是精確量測儀器

「SVV 不是『看到數值高就一定灌水』的指標，需放在整體血流動力學背景中判斷。」

SVV 解讀需結合通氣條件、心律、手術類型
ProAQT 監測參數速覽
參數說明臨床意義
CO / CI心輸出量 / 心臟指數整體循環功能評估
SV每搏輸出量前負荷與心肌收縮評估
SVV每搏輸出量變異容量反應性指標（需特定條件）
SVR / SVRI全身血管阻力後負荷評估
dPmx動脈壓力上升最大速率左心室收縮力代替指標（非純收縮力）
CPI心臟功率指數心臟整體做功能力，心因性休克評估
SVV 有效解讀的前提條件
必須滿足以下條件，SVV 才具有臨床參考價值
  • 正規心律（無心律不整）
  • 完全控制性機械通氣（無自主呼吸）
  • 足夠的潮氣量（建議 ≥8 mL/kg）
  • 胸腔未打開
  • 無嚴重 RV 功能障礙或肺動脈高壓
  • 無腹腔鏡手術（腹壓升高）
  • PEEP 不過高
MAP 低時的鑑別診斷決策樹
ProAQT 模式解讀處置方向
MAP↓ CI↓ SVV↑（條件有效）前負荷不足PLR 或小量 fluid challenge；觀察 SV/CI
MAP↓ CI↑/正常 SVR↓血管擴張性低血壓優先 Vasopressor
MAP↓ CI↓ SVV 不高 dPmx↓ CPI↓心肌收縮力降低Echo/TEE 確認；考慮 inotrope
MAP↓ SVR 極高 CI↓後負荷過高重新評估 vasopressor 劑量
SVV↑ 補液後 SV 不升非單純前負荷問題停止補液；評估 RV、肺壓、收縮力
容量反應性評估方法比較
方法可用於心律不整優點限制
PLR可可逆、不需補液需即時 SV/CO 量測；體位限制
EEO可（控制通氣）快速、可重複需深度鎮靜；無自主呼吸
Mini-fluid challenge可直接觀察 SV/CO 變化不可逆；液體過多風險
SVV / PPV不建議連續顯示RR interval 變異干擾
Fast-Flush Test 判讀
結果問題原因處置
1–2 次振盪後快速回正正常系統可信繼續配合臨床判斷
振盪 > 3 次阻尼不足氣泡、管路過長、共振排除氣泡、縮短管路
幾乎無振盪、回升緩慢過度阻尼氣泡、血塊、管路扭折檢查管路、回抽、排氣
SBP 高 DBP 低、脈壓寬阻尼不足（收縮壓過高）系統振動先校正再解讀
SBP 低 DBP 高、脈壓窄過度阻尼（訊號變鈍）血塊、氣泡、扭折檢查並清除
ProAQT 設定步驟口訣
備物完整 → 資料輸入 → 壓力歸零 → CI 校正 → 畫面設定
  • 備物完整：ProAQT 主機、傳輸線、電源線、A-line 導管、沖洗液（N/S + Heparin 500 IU）、壓力傳感器組
  • 資料輸入：身高、體重、出生日期（DD/MM/YYYY）、性別、成人/小兒
  • 壓力歸零：確認傳感器在腋中線第 4 肋間（靜液軸），開放大氣後按 Zero 鍵
  • CI 校正：選 ProAQT → Automatic CI → 確認後按 Calibrate 鍵
  • 畫面設定：至少顯示 MAP、CI/CO、SVV、SVR/SVRI、dPmx/CPI
動脈導管置入部位選擇
部位第一選擇時機優點缺點
橈動脈一般成人手術淺表、易壓迫、感染風險低嚴重休克時可能失敗
股動脈休克、末梢灌流差休克時較易操作，壓力更接近中心感染風險較高
肱動脈橈/股動脈均不可行時比橈動脈更接近中心靠近神經；無雙側循環保護
